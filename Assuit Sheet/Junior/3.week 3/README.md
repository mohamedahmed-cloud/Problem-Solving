# Sorted list & Sorted Set
- we Can use 
  - Sorted List
  - Sorted Set
  - Sorted Dict
in python using by download external module called `sortedcontainers`
```py
    from sortedcontainers import SortedList
    sl = SortedList(['e', 'a', 'c', 'd', 'b'])
    print(sl)
```
